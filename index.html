HTML Cheat Sheet

HTML Heading Tag

Tag Name Description
<h1> Heading 1 Defines the most important heading.
<h2> Heading 2 Defines a level 2 heading.
<h3> Heading 3 Defines a level 3 heading.
<h4> Heading 4 Defines a level 4 heading.
<h5> Heading 5 Defines a level 5 heading.
<h6> Heading 6 Defines the least important heading.

HTML Formatting Tags

Tag Name Description
<b> Bold Defines bold text.
<i> Italic Defines italic text.
<em> Emphasise Defines emphasized text.
<strong> Strong Defines strongly emphasized text.
<u> Underline Defines underlined text.

HTML Semantic Tags

Tag Name Description
<header> Header Represents introductory content or a group of navigational links.
<nav> Navigation Defines a set of navigation links.
<em> Emphasise Defines emphasized text.
<main> Main Represents the main content of the document.
<section> Section Defines a section in a document.
<article> Article Defines an independent piece of content within a document.
<aside> Aside Represents content tangentially related to the content around it.
<footer> Footer Represents a footer for a document or section.

HTML Link Tags

Tag Name Description
<a> Anchor Defines a hyperlink.
<link> Link Defines an external resource link.

HTML Table Tags

Tag Name Description
<table> Table Defines a table.
<caption> Caption Defines a table caption.
<thead> Table Head Groups the header content in a table.
<tbody> Table Body Groups the body content in a table.
<tfoot> Table Foot Groups the footer content in a table.
<tr> Table Row Defines a row in a table.
<th> Table Header Cell Defines a header cell in a table.
<td> Table Data Cell Defines a cell in a table.

HTML Form Tags

Tag Name Description
<form> Form Defines an HTML form for user input.
<input> Input Defines an input control.
<textarea> Textarea Defines a multiline input control.
<button> Button Defines a clickable button.
<select> Select Defines a drop-down list.
<option> Option Defines an option in a drop-down list.
<label> Label Defines a label for an input element.

HTML Media Tags

Tag Name Description
<img> Image Defines an image.
<audio> Audio Defines sound content.
<video> Video Defines a video.
<source> Source Defines multiple media resources for media elements.

HTML Script Tags

Tag Name Description
<script> Script Defines a client-side script.
<noscript> NoScript Defines content to be displayed if the browser does not support scripting.

HTML Meta Tags

Tag Name Description
<meta> Meta Defines metadata about an HTML document.
